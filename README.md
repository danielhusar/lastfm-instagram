last fm instragram stream [![Build Status](https://travis-ci.org/danielhusar/lastfm-instagram.png)](https://travis-ci.org/danielhusar/lastfm-instagram)
=========================

Install dependencies:
	
	npm install
	bower install
	
Build assest:

	gulp
	
Run the app by:

	node app
	
Run app with watch task to less files:

	gulp server

Run karma tests:

	npm test
